## Problem Statement
Organizations face major challenges in managing employee attendance, leave, and payroll using manual or fragmented systems. These methods cause errors, lack of transparency, and delay in approvals. A centralized, secure, and role-based system is required to streamline HR operations.

## Solution
Dayflow is a web-based Human Resource Management System (HRMS) that digitizes and automates employee management, attendance tracking, leave processing, and payroll visibility. The system provides role-based access for Employees and HR/Admin to ensure data security, transparency, and efficiency.

## Core Modules
- User Authentication (Employee / HR)
- Employee Profile Management
- Attendance Tracking
- Leave Management
- Payroll Viewing & Control
- Admin Approval Workflows

## Tech Stack
Frontend: HTML, CSS, JavaScript  
Backend: Node.js / Flask  
Database: MongoDB / MySQL  
Authentication: JWT / Sessions  
Version Control: GitHub
